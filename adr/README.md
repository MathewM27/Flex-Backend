# Architecture Decision Records

Light-weight records of the *why* behind non-obvious choices. One page each.
Numbered sequentially, never renumbered, never deleted. If a decision is
superseded, write a new ADR that links back ("Supersedes ADR-0003").

Add a new ADR by copying `template.md` and incrementing the number.
